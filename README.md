## onecircle 一个圈子

## 名字就叫一个圈子。
## version：1.1

### 想弄一个类社交圈子，开发了一部分，剩下慢慢折腾
(￢_￢)
目前特色有：
1. 前端仿推特
2. 支持前台直接提交，支持图文混合和链接（看下图）
3. 支持用户关注和取关
4. 支持后台自定义设置
5. 支持用户注册默认更改权限为贡献者，并自动通过审核，这样新注册的用户也能自己发推了
6. 支持给每个新注册用户添加个人签名
